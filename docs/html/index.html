Overview

The AV Catalog Converter is a specialized tool designed to standardize diverse audio-visual equipment catalogs into a consistent format. It uses a combination of pattern matching, heuristics, and machine learning to analyze, map, and transform catalog data from various manufacturers and suppliers into a standardized output format.

The application is built as a modular system with a Flask-based API backend and supports both command-line and web-based interfaces. It leverages the Microsoft Phi-2 model for intelligent field mapping and category extraction.
